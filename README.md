<h3>Protótipo</h3>
<p>Rascunho para explorar ideias  e validar hipóteses sem investir em desenvolvimento.</p>

<h4> Passos para prototipação</h4>
<ul>
  <li>Planejamento</li>
  <li>Rascunho</li>
  <li>Wireframe</li>
  <li>Protótipo de baixa fidelidade</li>
  <li>Protótipo de alta fidelidade</li>
  <li>Testes</li>
  <li>Iteração</li>
</ul>


<h1> Passos para prototipagem </h1>
1. Criar um novo projeto no <a href="http://figma.com">Figma</a>
2. Criar um guia de estilo.
3. Definir uma tela para prototipar.
4. No final da prototipação, criar a animação da tela.
